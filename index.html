Who I am

If you need graphic design and you need the result I will be glad to meet you. I am skilled in various design task.
As a web UI designer, create websites using HTML5 / CSS3 / jQuery. I start with developing layouts in Photoshop and implement it in HTML using modern frameworks such as Foundation, Pure, Bootstrap etc. I am familiar with responsive web design and make layouts look good in various platforms. I try to keep my code clean and user-friendly.
As an illustrator I work with Photoshop, Illustrator, and CorelDraw depending on the tasks. Each drawing starts with a sketch that I do by hand or on a graphic tablet. I also have experience in the development of print design: corporate identity, logo, brand books, etc.
